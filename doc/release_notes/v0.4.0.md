### Optional `etc/` Hook Examples

- Added `etc/env.sh` example hook
- Added `etc/aliases.sh` example hook
- Both files are examples and only take effect when hook loading is explicitly enabled

### Metadata Controls

Added extension metadata flags in `.extension`:

- `load_env: false`
- `load_aliases: false`

These defaults keep behavior safe and backward compatible.

### Opt-In Loading Model

Hook files are sourced only when both are true:

1. Global OraDBA setting: `ORADBA_EXTENSIONS_SOURCE_ETC=true`
2. Extension metadata flag: `load_env: true` and/or `load_aliases: true`

### Documentation and Tests

- Updated template documentation (`README.md`, `doc/configuration.md`, `doc/reference.md`)
- Extended tests to verify:
  - hook example files exist
  - hook files are included in build tarballs
